# Federal-Budget-Simulator
Federal Budget Allocation Simulator with a Projection Engine
The file relies on tkinter for the GUI, widgets, and filedialog, matplotlib for graphing, numpy for analysis and json for data storage
The application allows users to reallocate the 2025 United States Federal Budget between a variety of categories and see its social outcomes and GDP impact in a 10 year projection. 
The user is graded on their results and is allowed to save up to 2 scenarios and compare them to see how certain moves impact the nation in a way that is as close to realistic as possible.
